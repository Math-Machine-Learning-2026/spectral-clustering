# spectral-clustering
Spectral clustering para segmentación de imágenes
Creado por: Eva Gallego, Lucía Puche, Adrián Sánchez y Juan Mazo

Este repositorio contiene la implementación y el estudio analítico del algoritmo de Clustering Espectral para la segmentación no supervisada de imágenes complejas.
A diferencia de los modelos básicos, este enfoque utiliza la teoría de grafos y el álgebra lineal para aislar objetos con precisión, superando los retos clásicos de la visión artificial como sombras arrojadas, reflejos metálicos y oclusiones.

Se añaden al repositorio los siguientes elementos:
- El código fuente y los análisis están organizados en Cuadernos de Jupyter.
- Las imágenes utilizadas en los experimentos están almacenadas en la carpeta IMAGENES
- Documentos PDF en los que se da un marco teórico y se detalla la implementación del código para cada uno de los experimentos.

La estructura principal es la siguiente:
1. Notebook Principal(SpectralClustering_codigo_final.ipynb): mediante una imagen de ejemplo (frutas.png) se explica detalladamente el código
2. Juego y variaciones de parámetros: en los siguientes notebooks se hace un estudio de como los parámetros pueden variar el resultado aplicado a diferentes imágenes

Si se desea probar el código unicamente hay que almacenar las imágenes en la misma carpeta que los notebooks y ejecutarlos.
